## SPOTIFY POPULARITY SCORE SYSTEM

### The Two Scores
| Score Type | What It Is | Why It Matters |
|---|---|---|
| Artist PS | 0–100 on WHOiSEE's profile. Driven by catalog engagement, saves, search volume, follower growth. | Determines how large Spotify's test group is for each new release. |
| Track PS | 0–100 per ISRC. Driven by recent streams, saves, completion rate, skip rate. | Determines whether "Get Down" gets into Release Radar (20+) and Discover Weekly (30+). |

### Critical Threshold Map
| Score | What Unlocks | Timing |
|---|---|---|
| Track 20+ | Release Radar for non-followers | Release Radar refreshes FRIDAY |
| Track 30+ | Discover Weekly placement | Discover Weekly refreshes MONDAY |
| Artist 30+ | Test groups expand to 1,000+ per release | Cumulative — driven by cadence + saves + search |
| Artist 40+ | ~5,000 test listeners per release. Radio/mix playlist likely. | 3–6 months from current position |
