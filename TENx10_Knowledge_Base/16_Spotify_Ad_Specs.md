## SAVE LINK OPTIONS

### Option 1: Direct Spotify Link
`https://open.spotify.com/track/[TRACK_ID]`
- Pros: Direct to Spotify, no intermediary
- Cons: Only works for Spotify users

### Option 2: Smartlink (Bandsintown/Linkfire)
- Pros: Multi-platform (Spotify, Apple Music, YouTube, etc.)
- Cons: Extra click, slight conversion loss
- Use if you want to simultaneously push saves on Apple Music

### Option 3: Spotify Pre-Save Style Landing Page
- Create a simple page on dirtysnatcharecords.com that redirects to Spotify
- Pros: Pixel fires on your domain, then redirects to Spotify
- Cons: Requires building the page

**RECOMMENDATION:** Use Option 1 (direct Spotify link) for this campaign since the goal is specifically Spotify saves. Use the Bandsintown smartlink for organic social posts.
