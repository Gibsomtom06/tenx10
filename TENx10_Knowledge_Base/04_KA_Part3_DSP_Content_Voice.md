# MODULE 8: DSP ALGORITHMIC PLAYBOOK

## CRITICAL TIMING
The first 7 days after release determine permanent algorithmic placement across all platforms. The AI agent must trigger the release checklist automatically on release day and track completion.

## Core Grading System

### Track Quality Score (ISRC Level)
- **30-Second Rule:** High skip rate before 30 seconds = permanently shadow-demoted
- **Bucket Logic:** All streams/saves/shares for an ISRC go into one bucket. Changing ISRC resets to 0.
- **Threshold:** ISRC needs ~30+ out of 100 on Spotify to trigger algorithmic takeovers

### Artist Quality Score (Profile Level)
- **Active Search Weight:** #1 driver. Manual searches = "Destination Artist" status
- **Inherited Authority:** High Artist Score gives new ISRCs larger algorithmic test groups (1,000+ instead of 100)
- **Consistency Decay:** No new ISRC every 6-8 weeks = Artist Quality Score decays

### Algorithmic Test Group Sizes

| Artist Score | Test Group | What It Means |
|---|---|---|
| 0-20 | ~100 listeners | Track barely gets tested |
| 20-30 | ~500 listeners | Small test, needs strong signals |
| 30-40 | ~1,000 listeners | Moderate test, Discover Weekly possible |
| 40-60 | ~5,000 listeners | Strong test, Release Radar likely |
| 60-80 | ~10,000+ listeners | Major algorithmic push |
| 80-100 | ~50,000+ listeners | Viral-level distribution |

## Spotify — Release Week Actions

1. **Discovery Mode:** Toggle ON immediately. Trades ~30% royalty for 2-3x algorithmic impressions. Worth it for first 2-4 weeks.
2. **Canvas:** Upload 3-8 second looping video. Tracks with Canvas get 145% more shares. Within 24 hours.
3. **Clips:** Record 30-60 second commentary. Appears in Spotify feed.
4. **Save Campaign (Meta Ads):** $50-100 over 7 days targeting genre fans in tour markets. Target save-to-stream ratio > 10%.
5. **Editorial Pitch:** 7+ days BEFORE release via S4A. Include genre, mood, instruments, tour dates. Pitch is a STORY, not a description.
6. **Playlist Seeding:** Submit to independent curators via SubmitHub or direct outreach. Target 5-10 playlists.

### Key Metric: Save-to-Stream Ratio
If saves/streams > 10% → triggers Discover Weekly and Release Radar. AI monitors daily during week 1.

### Popularity Index Targets

| Current PI | Target | Action |
|---|---|---|
| 0-20 | 25+ | Heavy save campaign + Discovery Mode + playlist push |
| 20-35 | 40+ | Save campaign + Canvas + Clips + social push |
| 35-50 | 55+ | Maintain momentum, focus on editorial |
| 50+ | Maintain | Organic momentum. Monitor for decay. |

## Apple Music — Shazam Spike Strategy

Shazam is Apple Music's #1 external signal. Regional spike → "Trending on Shazam" → Apple editorial algorithm.

1. **Geo-Targeted Meta Ads:** $50-75 in ONE city (next tour stop) 3-5 days before show. Video views objective (ThruPlay). People hear track → Shazam it.
2. **Artist Message:** Voice note through Apple Music for Artists. Within 48 hours of release.
3. **Lyrics Sync:** Confirm in Musixmatch for "Hey Siri" voice search matching.

### Shazam Spike Campaign Template

| Parameter | Value |
|---|---|
| Platform | Meta Ads Manager (Instagram/Facebook) |
| Objective | Video Views (ThruPlay) — NOT conversions, NOT traffic |
| Budget | $50-75 per city |
| Duration | 3-5 days before show date |
| Targeting | Age 18-34, bass music/dubstep/EDM interests, geo-locked to single metro (25mi radius) |
| Creative | 12-15 second vertical video with track audio prominently featured |
| Placements | Instagram Reels, IG Stories, Facebook Stories (sound-on environments) |
| CTA | None — goal is Shazam recognition, not clicks |
| Exclusions | Exclude existing followers (they already know) |
| UTM | utm_source=meta, utm_medium=paid, utm_campaign=shazam_spike_{city} |

## TikTok — Audio Virality

2. Make audio available as a Sound — #1 signal. Other users using your audio = trending sound.
3. Ask support artists/friends to post using the audio. 5-10 videos in week 1 can trigger trending.
4. Engage with comments aggressively in first 2 hours. TikTok algorithm weighs early engagement heavily.

# MODULE 9: TOURING PHASE SYSTEM

## Phase Definitions & Auto-Transitions

| Phase | Timeline | Trigger | AI Agent Actions |
|---|---|---|---|
| ANNOUNCEMENT | 4-8 weeks before show | Show confirmed, contract signed | Generate announcement content (3-5 posts). Launch Shazam Spike if in budget. Add to touring grid. |
| ON-SALE | 3-5 weeks before show | Tickets go live | Generate ticket push content. Track ticket velocity. Launch retargeting if pixel set up. |
| FINAL PUSH | 7-10 days before show | Auto-triggered by date | Generate urgency content ("7 DAYS", "THIS FRIDAY"). Daily posts. Increase ad spend. Push ticket link in every post. |
| MAINTENANCE | Post-announcement, low activity | Show confirmed but > 4 weeks out | Periodic reminders (1-2/week). No heavy ad spend. Focus budget on closer shows. |

## Phase Transition Rules

```
Show date minus current date:
    > 28 days → ANNOUNCEMENT or MAINTENANCE
    21-28 days → ON-SALE (if tickets live)
    7-10 days → FINAL PUSH (auto-trigger urgency content)
    0-6 days → FINAL PUSH — MAXIMUM URGENCY
    Past date → POST-SHOW → Trigger promoter/venue grading
```

## Content Volume by Phase

| Phase | Posts/Week | Ad Spend | Content Type |
|---|---|---|---|
| Announcement | 2-3 | $0-25 | Flyer, "just announced" post, venue hype |
| On-Sale | 3-4 | $25-50 | Ticket link posts, lineup graphics, BTS |
| Final Push | Daily | $50-75 | Countdown, urgency captions, last call, ticket giveaways |
| Maintenance | 1-2 | $0 | Casual mentions, show list updates |

## 5-Phase National Marketing Calendar (From Tour Book)

| Phase | Timing | Focus |
|---|---|---|
| Phase 1: Announcement Week | T-12+ weeks | Full tour flyer rollout, asset pack to promoters, press release |
| Phase 2: Early-Bird Push | T-11 to T-9 | Build momentum, fast ticket movement, plant narrative |

# MODULE 10: CONTENT & ASSET ENGINE

## Asset Management System

### Asset Upload & Storage
Users can upload or link assets that the AI agent references when generating content:

**Asset Types:**
- **Photos:** Press photos, live show shots, behind-the-scenes, venue photos
- **Videos:** Live set clips, studio sessions, talking head reels, tour recap footage
- **Graphics:** Tour flyers (per-city versions), album artwork, countdown templates, brand graphics
- **Audio:** Track snippets (15s, 30s, 60s clips), drops, intros for ad creative
- **Logos:** Artist logo (PNG, SVG, white/black/transparent), label logo
- **Templates:** Flyer templates (Canva/Photoshop), story templates, countdown templates

**Upload Methods:**
1. Direct upload to platform storage (Supabase Storage)
2. Link to Google Drive folder (syncs automatically)
3. Link to Dropbox or external URL
4. Capture from social media (link an Instagram post or TikTok as a reusable asset)

**Asset Tagging:**
Every asset gets tagged with:
- Type (photo, video, graphic, audio, logo, template)
- Artist (which artist on roster)
- Show (linked to specific show if applicable)
- Release (linked to specific release if applicable)
- Platform suitability (Instagram feed, IG Stories, TikTok, YouTube Shorts, Facebook, Twitter)
- Dimensions (1080x1920 vertical, 1200x1200 square, 1920x1080 landscape)

### Asset-to-Content Pipeline

```
ASSET LIBRARY → AI selects appropriate assets based on:
    1. Current phase of the show being promoted
    2. Platform being posted to (vertical for TikTok/Reels, square for IG feed)
    3. Content type (energy clip needs video, announcement needs flyer graphic)
    ↓
AI GENERATES CAPTION → Using voice profile (Module 11) + show data + phase rules
    ↓
CONTENT PACKAGE OUTPUT:
    - Caption text (platform-specific)
    - Suggested asset(s) from library with file reference
    - Hashtag set (rotated from pre-built sets)
    - Posting time recommendation
    - CTA (one per post: tickets, save track, tag crew)
    ↓
APPROVAL QUEUE → Content enters approval queue
    - APPROVED → auto-schedules or marks ready-to-post
    - REVISION → AI regenerates with feedback
    - SKIP → ignored
```

### Content Script Generator

For video content (Reels, TikToks, YouTube Shorts), the AI generates scripts:

**Script Template: Talking Head (15-30 sec)**
```
[HOOK — first 3 seconds, most important]
"{city} THIS {day_of_week}." or "yo what's up" or "new music alert"

[BODY — 10-20 seconds]
Brief hype about show/release + call out venue or support artists

[CTA — last 3-5 seconds]
"tickets in bio" or "go save it on Spotify" or "tag your crew"

[TEXT OVERLAY]
{city} · {date} · {venue}
{artist_name}

[AUDIO]
Use {latest_release_title} as background audio
```

**Script Template: Energy Clip (12-15 sec)**
```
[VISUAL] Quick cuts of live show footage — crowd, lasers, bass drops
[AUDIO] Hardest 12-15 seconds of {track_title}
[TEXT OVERLAY] 
    Slide 1 (0-3s): "{city} {day_of_week}" 
    Slide 2 (3-10s): no text, let the energy hit
    Slide 3 (10-15s): "tickets in bio 🔗"
```

**Script Template: Shazam Spike Ad (12-15 sec)**
```
[VISUAL] Dark screen → glitch effect → bass drop hits → crowd footage
[AUDIO] Most identifiable section of {track_title} — do NOT show track name
[TEXT OVERLAY]
    "WHAT TRACK IS THIS? 🛸"
    "Shazam it."
[END CARD] {artist_name} @ {venue} · {date} · {city}
[RULE] No track title shown — force the Shazam
```

## Platform-Specific Rules

### Instagram
- Feed: High-quality graphics or professional photos. Caption under 3 sentences. City + venue + date + link in bio.
- Stories: 3-5/day on show weeks, 1-2 on off days. Use countdown sticker, poll sticker, link sticker.
- Reels: 15-60 sec, vertical, high energy. Use latest release audio when possible.

### TikTok
- SEO title is everything. Write it like a YouTube title.
- First 3 seconds must hook. Drop the heaviest part immediately.
- Post minimum 2x/week to stay in rotation. 1-2x daily during Final Push.
- Make audio available as Sound for re-use.

### YouTube Shorts
- Cross-post best Reels/TikToks.
- Include track title in description for audio linking.
- SEO titles matching TikTok strategy.

### Twitter/X
- Short, punchy. One sentence + link.
- Game day tweets with venue tags.
- Quote-retweet fan content from shows.

### Facebook
- Create Facebook Events for every show (feeds Bandsintown and Google).
- Longer captions acceptable. Include all show details.
- Share to genre-specific groups.

# MODULE 11: VOICE PROFILE SYSTEM

## What Is a Voice Profile?

Every artist on the platform gets a voice profile that defines how the AI writes content in their voice. This is NOT optional — the AI must match the artist's tone or the content feels corporate and fake.

## Voice Profile Fields

```json
{
  "voice_profile_id": "uuid",
  "artist_id": "uuid",
  "tone": "string — e.g. 'hype, raw, authentic, unapologetic'",
  "language_style": "string — e.g. 'casual, uses profanity naturally, slang'",
  "emoji_patterns": ["🔥", "🙏", "🛸", "👽", "‼️"],
  "capitalization": "string — e.g. 'mixes caps for emphasis, brand name always stylized'",
  "caption_length": "string — e.g. 'short, punchy, rarely long paragraphs'",
  "brand_catchphrase": "string | null — e.g. 'PLAY SOME F*CKING DUBSTEP'",
  "words_to_use": ["string — words the artist actually uses"],
  "words_to_avoid": ["string — words that feel wrong for this artist"],
  "example_captions": [
    "string — 3-5 real captions the artist has posted"
  ],
  "exception_notes": "string — e.g. 'longer heartfelt posts for milestones like Lost Lands'"
}
```

## How Voice Profiles Are Built

1. **During onboarding:** AI asks "Paste 3-5 of your best social media captions" or "Link your Instagram so I can study your voice."
2. **From OAuth data:** If Instagram is connected, AI analyzes last 20 posts for tone, length, emoji usage, capitalization patterns.
3. **Manual tuning:** User can edit the voice profile anytime: "I want the AI to sound more hype" or "never use the word 'incredible'"
4. **Learning over time:** If user consistently edits AI-generated captions the same way (e.g., always shortening them), the voice profile auto-adjusts.

## Caption Generation Rules (Universal)

- **Voice:** Match the artist's voice profile. Bass music artists are not corporate.
- **Length:** Instagram captions: 1-3 sentences max. Twitter: 1 sentence + link. TikTok: SEO-optimized title, not a caption.
- **CTA:** Every post needs ONE clear call-to-action. Not three. Either "grab tickets," "save the track," or "tag your crew." Pick one.
- **Urgency words (Final Push):** "THIS FRIDAY," "LAST CALL," "almost sold out," "don't sleep," "link in bio NOW"
- **AVOID:** "We're so excited to announce..." (too corporate). "Don't miss out on this incredible..." (too salesy). Generic emojis without context.
- **INCLUDE:** City name in every show post. Venue name when confirmed. Ticket link or "link in bio." Tag venue and promoter.
